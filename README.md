# Deploy TerminusDB to Heroku

Click this button, and follow instructions there, it will ask you for an appname, this neeeds to be a lowercase word without spaces, you will also need to set the root password.

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

Once TerminusDB is unstalled you will be able to login with theurl for your heroku app, i.e. https://terminusdb.herokuapp.com and the password you set.
